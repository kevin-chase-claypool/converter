XY Theta Plotter Electronics Layout and Wiring Concept

Current planning view for the RP23CNC/grblHAL controller, three TB6600 stepper channels, toolhead electronics, Ethernet control, and the TMAG5273/RP2040 magnetic homing adapter.

This is an explanatory layout, not an authority for terminal labels or wire colors. The master wiring table controls. Every dashed connection remains TBD until the matching inspection or bench test passes.

Current Control Flow

  • Host PC sends G-code and jog commands to RP23CNC over USB first, Ethernet after W5500 bring-up.
  • RP23CNC/grblHAL owns X/Y/A motion, normal homing, limits, and M3/M5 output.
  • RP2040 reads the TMAG5273 and reports magnetic data to the host calibration script.
  • The optional RP2040 A_HOME output must behave like a verified switch input before connection.

Current Power Concept

  • AC mains feeds the MEISHILE S-120-12 through a protected inlet, switch, fuse, enclosure, and strain relief that are still TBD.
  • The 12 V bus feeds RP23CNC, X/Y/A TB6600 drivers, and the toolhead buck through fused branches still TBD.
  • The B085T73CSD buck is planned for 6.0 V toolhead actuator power after E-14/E-15 verification.
  • Protective earth and 0 V return routing must be verified before power is applied.

Homing and Calibration

  • X/Y home from physical limit switches wired to RP23CNC.
  • The center cylindrical magnet is scanned inside a bounded 4 in x 4 in area to locate bed center.
  • The outer cylindrical magnet near 8.9 in radius provides the theta/A angular index.
  • Saturated magnetic fields are handled by finding opposing footprint edges, not by looking for a peak.
